🆘 QUICK TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

PROBLEM: Extension not showing
SOLUTION: 
  1. Reload VS Code: Ctrl+Shift+P → "Reload Window"
  2. Check extension is installed: Extensions panel
  3. Make sure file is .ini
  4. Make sure you're in a workspace (open folder)

PROBLEM: Timing data not loading
SOLUTION:
  1. Check file exists: ls layer_timing_profile.txt
  2. Check file location (same folder as .ini)
  3. Check file format (space-separated columns)
  4. Check VS Code console for errors: Ctrl+Shift+I

PROBLEM: Wrong timing file name
SOLUTION:
  1. Edit extension_complete.js line ~55
  2. Change 'layer_timing_profile.txt' to your name
  3. Rebuild/reinstall extension

PROBLEM: Need more help
SOLUTION:
  → Read INSTALLATION_GUIDE.md → Troubleshooting section
